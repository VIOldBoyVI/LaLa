# ЛА-ЛА-ГЕЙМ - Викторина с элементами караоке (с сохранением состояния)

ЛА-ЛА-ГЕЙМ — веб-игра-викторина с элементами караоке. Игра выполнена в виде веб-приложения с интерактивным полем 10×10, где игроки открывают ячейки, отвечают на вопросы и зарабатывают очки.

## Особенности игры

- Игровое поле 10×10 с буквенной и числовой адресацией
- 5 раундов: 4 раунда по 20 вопросов и 1 финальный с 10 вопросов
- Перерывы после 2-го и 4-го раундов (по 10 минут)
- Каждый раунд имеет свою тему (музыкальные жанры, исполнители, инструменты и т.д.)
- Система подсчёта очков
- Сохранение состояния игры между сессиями
- Адаптивный дизайн для настольных компьютеров

## Требования

- Python 3.6 или выше
- Flask
- SQLite (обычно входит в стандартную библиотеку Python)

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

1. Запустите приложение:
   ```bash
   python run.py
   ```
2. Откройте в браузере: http://127.0.0.1:5555

## API endpoints

- `POST /api/init_game` - инициализация новой игры
- `POST /api/get_question` - получение вопроса для раунда
- `POST /api/check_answer` - проверка ответа на вопрос
- `POST /api/save_state` - сохранение состояния игры
- `GET /api/load_state` - загрузка состояния игры

## Игровой процесс

1. Игроки открывают ячейки на игровом поле
2. При открытии ячейки с номером появляется вопрос
3. При правильном ответе игрок получает 10 очков
4. После 20 открытых ячеек в раундах 2 и 4 начинается 10-минутный перерыв
5. Всего 5 раундов, после которых игра завершается
