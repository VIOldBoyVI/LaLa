# ЛА-ЛА-ГЕЙМ

ЛА-ЛА-ГЕЙМ — веб-игра-викторина с элементами караоке. Игра выполнена в виде одностраничного веб-приложения с интерактивным полем 10×10, где игроки по очереди открывают ячейки, отвечают на вопросы и зарабатывают очки.

## Особенности игры

- Игровое поле 10×10 с буквенной и числовой адресацией
- 5 раундов: 4 раунда по 20 вопросов и 1 финальный с 10 вопросов
- Перерывы после 2-го и 4-го раундов (по 10 минут)
- Каждый раунд имеет свою тему (музыкальные жанры, исполнители, инструменты и т.д.)
- Система подсчёта очков
- Сохранение состояния игры между сессиями
- Адаптивный дизайн для настольных компьютеров

## Требования

- Python 3.6 или выше
- Flask
- SQLite (обычно входит в стандартную библиотеку Python)
